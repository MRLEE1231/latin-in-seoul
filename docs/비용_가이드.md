## 무료로 시작하는 방법 (완전 무료 구성)

```
프론트엔드 + 백엔드: Next.js (무료)

배포: Vercel 무료 티어 (무제한)
```

**예상 비용: $0/월** 💰

**제한사항:**
- 데이터베이스: 500MB 저장공간 (수천 개 게시글 가능)
- 트래픽: Vercel 100GB/월 (소규모 프로젝트 충분)

**적합한 경우:**
- 개인 프로젝트
- 소규모 서비스
- MVP/프로토타입
- 학습 목적

## 비용 발생 시점

### 언제 비용이 발생하나요?

1. **데이터베이스 용량 초과**
   - Supabase 무료: 500MB 초과 시
   - 해결: 유료 플랜 업그레이드 ($25/월)

2. **트래픽 증가**
   - Vercel 무료: 100GB/월 초과 시
   - 해결: Pro 플랜 ($20/월)

3. **이미지 저장 공간 증가**
   - 로컬 저장: 서버 용량 초과 시
   - 해결: Cloudflare R2 또는 S3 사용

## Oracle vs PostgreSQL 비용 비교

### Oracle (기존 스킬)
- **라이선스**: 상용 라이선스 필요 (수백만원/년)
- **호스팅**: Oracle Cloud (무료 티어 있지만 제한적)
- **개인 프로젝트**: 비용 부담 큼

### PostgreSQL (Next.js 추천)
- **라이선스**: 완전 무료 (오픈소스)
- **호스팅**: Supabase 무료 티어 (500MB)
- **개인 프로젝트**: $0로 시작 가능

**결론**: PostgreSQL이 훨씬 경제적입니다.
