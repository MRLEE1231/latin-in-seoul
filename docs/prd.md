# Latin in Seoul — 프로젝트 기획서

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| **프로젝트명** | Latin in Seoul |
| **목적** | 라틴 댄스 수업 정보를 소개·공유하는 웹앱 |
| **핵심 가치** | 수업 게시판을 통한 수업 정보 탐색 및 공유 |

---

## 2. 핵심 기능

### 2.1 수업 게시판

- **역할**: 라틴 댄스 수업 정보글 목록 제공
- **동작**: 게시글 목록 조회 → 개별 게시글(수업 정보글) 상세 보기

### 2.2 수업 정보글 (게시글 단위 콘텐츠)

한 게시글 = 한 수업 정보 단위. 아래 필드를 가짐.

| 필드명 | 설명 | 타입/제약 |
|--------|------|-----------|
| **이미지** | 수업/장소/분위기 사진 | 다중 이미지 업로드 지원 |
| **텍스트** | 본문 설명 (자유 텍스트) | 필수 |
| **작성일자** | 글 작성일 | 날짜 (자동 또는 수동) |
| **지역** | 수업 장소 지역 | 강남 / 홍대 / 그외 (선택) |
| **종류** | 댄스 종류 | 살사 / 바차타 / 키좀바 (선택) |
| **강사명** | 담당 강사 이름 | 텍스트 |
| **수업설명** | 수업 내용·시간·비용 등 상세 설명 | 텍스트 (필수 또는 권장) |

---

## 3. 데이터 구조 (안)

### 3.1 수업 정보글 엔티티

```
게시글 (Post / ClassInfo)
├── id
├── 이미지 URL[] (또는 첨부파일 목록)
├── 본문 텍스트
├── 작성일자 (createdAt)
├── 지역 (region: GANGNAM | HONGDAE | ETC)
├── 종류 (danceType: SALSA | BACHATA | KIZOMBA)
├── 강사명 (instructorName)
├── 수업설명 (classDescription)
└── (선택) 수정일자, 작성자 등
```

### 3.2 지역·종류 코드

- **지역**: `강남`, `홍대`, `그외`
- **종류**: `살사`, `바차타`, `키좀바`

---

## 4. 화면 구성 (안)

| 화면 | 설명 |
|------|------|
| **목록 (게시판)** | 수업 정보글 리스트 (카드/리스트), 지역·종류 필터 가능 권장 |
| **상세 (정보글)** | 이미지 갤러리 + 텍스트 + 작성일자, 지역, 종류, 강사명, 수업설명 표시 |
| **작성 (관리자)** | 이미지 업로드, 텍스트·지역·종류·강사명·수업설명 입력 폼 (구현 범위에 따라 추가) |

---

## 5. 우선순위 및 범위

| 순서 | 기능 | 비고 |
|------|------|------|
| P0 | 수업 게시판 목록 + 수업 정보글 상세 보기 | 필수 |
| P0 | 정보글 데이터: 이미지, 텍스트, 작성일자, 지역, 종류, 강사명, 수업설명 | 필수 |
| P1 | 지역/종류 필터, 검색 | 권장 |
| P2 | 관리자용 글 작성·수정·삭제 | 운영 시 필요 |

---

## 6. 기술 검토 시 고려사항

- **이미지**: 다중 업로드, 저장소(로컬/클라우드), 썸네일/원본 구분
- **지역·종류**: 고정 옵션 → 드롭다운/칩 선택 UI
- **작성일자**: 서버/클라이언트 타임존, 표시 포맷(예: 2025.02.12)
